# SlumLink
SLUMLINK is a smart digital platform for urban slum management. It enables household registration, health and aid tracking, text/voice complaint reporting with categorization, event management, and real-time dashboardsâ€”helping NGOs and authorities deliver services transparently and efficiently.

## Run locally (localhost)

Requirements: Node.js (LTS) + npm

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

The production build output is in the `dist/` folder.
